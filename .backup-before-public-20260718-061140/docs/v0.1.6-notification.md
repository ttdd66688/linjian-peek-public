# 掌心窗 v0.1.6 · 通知提醒版

在 v0.1.5 合并版基础上新增轻通知提醒能力。

## 新增功能

- Android 端新增 `send_notification` 执行动作。
- MCP 端新增 `send_notification(title, message, device_id)` 工具。
- App 生活模块新增“本机测试：弹出通知提醒”按钮。
- 新增通知渠道“掌心窗提醒”。Android 13+ 会请求通知权限。

## 使用方式

示例：

```text
send_notification(title="掌心窗提醒", message="你喝水，记得看一眼。")
```

如果手机没有弹出通知，请检查：

1. 系统设置里是否允许《掌心窗》发送通知；
2. App 是否已启动前台服务；
3. Render 的 server 与 MCP 是否都已重新部署到 latest commit；
4. 手机端是否正在正常轮询。
