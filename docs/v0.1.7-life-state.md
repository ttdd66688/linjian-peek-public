# 掌心窗 v0.1.7 生活状态层

本版新增 Life State：

- 电量百分比、充电状态、充电类型。
- 网络类型：wifi / cellular / ethernet / none。
- 屏幕亮灭。
- 当前前台 App 与包名。
- 今日屏幕使用时长、今日解锁/亮屏次数、最近一次解锁时间。
- 今日 Top App 使用列表。
- 城市与天气备注：先手动填写，后续可接天气 API。
- `accessibility_ready`、`usage_permission_ready`。

## 使用前授权

1. 设置页打开「无障碍设置」，开启掌心窗。
2. 设置页打开「使用情况访问权限」，允许掌心窗读取使用统计。
3. Android 13+ 允许通知权限。
4. 启动掌心窗服务。

## 接口

手机端随轮询上传：

```text
POST /api/device/state
```

GPT / MCP 读取：

```text
GET /api/life_state?device_id=my-phone
```

MCP 工具：

```text
get_life_state
```

## 设计原则

生活状态层默认不截图。能用“电量/当前 App/屏幕时间/解锁次数”表达关心时，就不必打开屏幕内容。
