# v0.1.5 合并说明

## 合并内容

1. App 统一叫「掌心窗」。
2. 保留截图能力：无障碍截图、后台轮询、上传 `/api/screenshot`。
3. 加入控制能力：打开 App、返回、主页、最近任务、点击、滑动。
4. 加入状态读取：当前包名、无障碍是否可用、屏幕文字摘要。
5. 加入闹钟命令：`set_alarm`，通过 `AlarmClock.ACTION_SET_ALARM` 调用系统闹钟。

## 闹钟说明

`set_alarm` 参数：

```json
{
  "hour": 8,
  "minute": 30,
  "message": "你起床，提醒时间到了",
  "vibrate": true,
  "skip_ui": true
}
```

注意：`skip_ui` 是请求跳过界面，不保证所有系统闹钟 App 都完全执行。有些手机会打开闹钟确认页。
