# v0.2.3 评论模式

本版在 v0.2.2 的精准点击能力上，新增小红书评论的双模式：

- `manual` 手动发送：打开评论框并写入草稿，不点击发送，用户自己检查后手动发送。
- `auto` 自动发送：写入评论时自动追加署名，例如 `（AI助手发）`，随后点击“发送”。

新增 MCP 工具：

- `xhs_comment`
  - `text`：评论正文。
  - `mode`：`manual` 或 `auto`。
  - `author_tag`：自动发送时追加的署名，默认 `（AI助手发）`。

示例：

```json
{"text":"这个标题真的有点茶哈哈哈","mode":"auto","author_tag":"（AI助手发）"}
```

说明：自动发送模式适合双方已经约定“由AI助手直接评论但注明来源”的场景；如果需要更稳妥，就使用 manual 模式只写草稿。
