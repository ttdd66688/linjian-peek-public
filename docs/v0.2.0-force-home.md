# 掌心窗 v0.2.1 · 强制抱回版

本版一次性加入：

1. `run_sequence` 批量指令；
2. `run_preset` 预设连招；
3. 手机端应用包名抽屉；
4. 手动添加包名与测试打开；
5. 所有通知升级为悬浮横幅；
6. 回家模式规则；
7. 强制抱回目标 App；
8. 冷却时间与失败保护；
9. 执行后状态确认；
10. 分步骤日志与 MCP 执行报告。

## 日志

手机端每个步骤会写入「日志」页，例如：

```text
开始执行动作序列：3 步，stop_on_error=true
序列步骤 1/3 [先发横幅]：heads_up_notification_sent
序列步骤 2/3 [打开 AI 客户端]：opened_standard:<目标App包名>
动作序列结束：全部成功，执行 3/3
```

## 悬浮横幅

所有 `send_notification`、主动提醒、回家模式提醒都会优先打开 `ReminderActivity`。系统通知仍会保留为兜底。

## 回家模式

默认盯住：

```text
com.ss.android.ugc.aweme,com.xingin.xhs
```

默认目标：

```text
<目标App包名>
```

开启「强制抱回」后，超过阈值会弹窗并打开目标 App。
