# 掌心窗 v0.1.6 UI 修订版

本版在 v0.1.6 通知提醒版基础上调整 UI：

- 底部导航栏：设置 / 看见 / 控制 / 生活 / 日志。
- 设置页只放服务器、Token、设备 ID、轮询间隔、无障碍和启动停止。
- 看见页放截图测试。
- 控制页放打开小红书、回桌面、返回。
- 生活页放闹钟和通知测试。
- 日志页单独展示调试日志。

同时确认保留 v0.1.5 后续修复：

- GitHub Actions 使用 setup-android 且只安装 platform-tools、platforms;android-34、build-tools;34.0.0。
- CompanionService 的 x2/y2 参数已补齐。
- peek 命令的 executeCommand 调用已补齐 x2/y2 两个 0。
