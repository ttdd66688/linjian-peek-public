# v0.2.2 精准点击与评论草稿

这一版把「看见标题再点击」从盲猜坐标升级为无障碍节点定位：

- Android 端会收集当前屏幕节点：文字、描述、控件类型、可点击状态、bounds、center。
- MCP 新增 `get_screen_nodes` 读取节点列表。
- MCP / 手机端新增 `tap_text`，可按文字包含/精确匹配点击。
- 新增 `input_text`，可向当前输入框输入文字。
- 新增 `draft_xhs_comment`，只写入评论草稿，不自动发送。
- 新增 `send_visible_comment_after_confirmation`，仅在用户明确确认后点击“发送”。

注意：小红书在部分荣耀手机上需要保留后台，冷启动可能停在桌面；这是系统/应用限制，不是连招失败。
