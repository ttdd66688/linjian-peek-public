# 掌心窗 v0.1.8 主动提醒层

v0.1.8 在 v0.1.7 的 Life State 生活状态层上新增 Active Reminder 主动提醒层。

## 设计目标

- 不依赖每次 GPT 手动发通知。
- 由手机端本机根据生活状态判断是否提醒。
- 每条规则有冷却，避免重复弹窗。
- 提醒文案偏生活陪伴，不做冷冰冰的系统告警。

## 已有规则

### 低电量

默认开启。

触发条件：`battery_percent <= threshold` 且未充电。

默认阈值：20%。

冷却：2 小时。

### 今日屏幕时长

默认开启。

触发条件：`screen_time_today_minutes >= threshold`。

默认阈值：240 分钟。

冷却：每天一次。

### 喝水提醒

默认关闭。

开启后按间隔提醒。

默认间隔：120 分钟。

首次启动静默计时，不会一打开就弹。

### 休息眼睛

默认开启。

屏幕亮着时按间隔提醒。

默认间隔：90 分钟。

首次启动静默计时。

### 生理期提醒

默认关闭。

填写：

- 上次开始日期 `yyyy-MM-dd`
- 平均周期天数
- 经期天数
- 提前几天提醒

状态层会返回：

- `cycle_enabled`
- `last_period_start`
- `next_period_start`
- `days_until_next`
- `is_period_now`
- `period_day`
- `current_phase`

若正在经期或距离预计开始时间小于等于提前提醒天数，每天提醒一次。

## 使用建议

1. 安装新 APK。
2. 开启无障碍权限。
3. 开启使用情况访问权限。
4. Android 13+ 允许通知权限。
5. 在状态页配置主动提醒规则。
6. 启动服务。

如果通知没有弹出，先检查系统通知权限与掌心窗 App 通知开关。
