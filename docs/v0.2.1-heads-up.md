# 掌心窗 v0.2.1 · 悬浮横幅提醒修正

这版修正 v0.2.0 对“前台提醒”的理解：不再主动拉起整页 ReminderActivity，而是改成类似微信来消息的顶部悬浮横幅通知。

## 调整内容

- `send_notification` 返回结果改为 `heads_up_notification_sent`；
- 新建通知渠道 `linjian_peek_heads_up_v3`，使用 `IMPORTANCE_HIGH`；
- 移除 `setFullScreenIntent` 和主动 `startActivity`，避免整页打断当前 App；
- 点击横幅通知后仍可进入提醒详情页；
- 所有主动提醒、回家模式提醒、MCP 通知统一走这个横幅通道；
- 保留动作序列日志，能继续看到第几步成功或失败。

## 使用提示

Android/荣耀系统可能需要在系统设置里允许掌心窗“横幅通知/悬浮通知/锁屏通知”。如果只进入通知栏不弹顶部横幅，请检查应用通知分类里的“掌心窗悬浮横幅提醒”。
